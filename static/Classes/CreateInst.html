Creating Instances


A class decleration creates a type name which makes it possible to declare variables of that type, also known as an instance of the class. First we declare the class Point.




To create new instances the "new" keyword is not used as is common in object-oriented programming languages, since it does not exist in Modelica. In MyPoints three instances of Point are created.




MyPointsInst creates three new Points by instantiating MyPoints. There are two different ways to give the variables initial values when creating the instance as shown in MyPointsInst and MyPointsInst2.
